## 🔬 Featured Projects

### 🏦 [Tazama Alert Navigator & Rule Trigger Discovery](https://github.com/shikha-pamnani/Tazama-X-Delta-Investigative-Visualization---Python-Project)
**Delta Analytics Fellowship · Linux Foundation · Gates Foundation · 2026**

An investigator-facing analytics tool built on an open-source AML/fraud detection platform deployed by financial service providers across South African markets.

- Built with **PySpark, Apache Hudi, Plotly, ipywidgets, Voilà** on JupyterLab
- Integrates **7 data sources** from a real-time Data Lakehouse
- Implements **15 of 33 fraud detection rules** — covering transaction convergence, account dormancy, purpose code classification, age classification, and regulatory threshold detection
- Surfaces EFRUP status, typology scores, rule weights, independent variables, and full transaction history for each alert
- Allows investigators to **immediately identify the specific behavior** that triggered each rule — eliminating manual search through entire transaction histories

`PySpark` `Apache Hudi` `Python` `Plotly` `JupyterLab` `AML` `Fraud Detection` `Financial Inclusion`
